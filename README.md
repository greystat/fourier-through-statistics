# fourier-through-statistics
A 140-page treatment of Fourier series that develops the theory through a probabilistic and statistical lens. Fejér's theorem is a statement about convergence in probability; Parseval's identity is an ANOVA-type variance decomposition. The chapter develops naturally, and intends to require minimal prerequisites beyond introductory calculus.
